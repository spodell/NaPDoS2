PKS/NRPS Domain Search

Choose a domain and query type, then enter your data to identify candidate KS and/or C domains.
Optional Advanced Settings can be used to customize program parameters.

Domain type

    KS domains
    C domains

Query type

    Amino acid sequences
    Nucleic acid sequences

Note: Unassembled Illumina sequencing reads from genomes and metagenomes and metagenomes are usually too short to obtain accurate PKS and NRPS domain predictions. Minimum DNA sequence lengths of at least 650 nucleotides are recommended, allowing sequence alignments to cover least 50% of verified PKS and NRPS domains, which are typically about 1250 nucleotides (425 amino acids) long.

Query sequence

Enter or paste sequence(s) in FASTA format, or upload a FASTA file. Please avoid using sequence id numbers containing non-standard characters (e.g. commas, slashes, colons, parentheses, ampersands, etc), as well as ambiguity codes "R" and "Y" for nucleic acid sequence files.

Note: Current size limit for uploaded files is 500 MB. For larger files, please consider our suggestions for file size management before submission. Large nucleotide sequence files may take several minutes to load.
